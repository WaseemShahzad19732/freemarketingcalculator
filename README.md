# freemarketingcalculator.com

Free marketing calculators for ROI, budget, conversion rate, salary, and more.

Built with [Astro](https://astro.build) — deploys to Cloudflare Pages.

## Calculators
- Marketing ROI Calculator (Basic ROMI + COGS variant)
- Marketing Budget Calculator (12 industry benchmarks)
- Conversion Rate Calculator
- Email Marketing ROI Calculator
- Social Media Budget Calculator
- Digital Marketing Calculator (CPM, CPC, CTR, ROAS, CPA)
- Marketing Salary Calculator
- Marketing Agency Valuation Calculator
- Marketing Automation ROI Calculator
- Inbound Marketing Calculator

## Dev
```bash
npm install
npm run dev      # localhost:4321
npm run build    # builds to dist/
```

## Deploy
Push to main → Cloudflare Pages auto-deploys.
- Build command: `npm run build`
- Output directory: `dist`
